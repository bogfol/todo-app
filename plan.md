# Accountability Feature Plan

## Low Scope Features

- Add task priority labels such as Low, Medium, and High.
- Add due dates to tasks so users know what needs attention soon.
- Show a daily task counter for completed and unfinished tasks.
- Add a "focus task" marker so one task can be highlighted as the current priority.
- Add confirmation before deleting a task to prevent accidental removal.
- Save tasks in local storage so progress is not lost after refreshing.

## Medium Scope Features

- Add recurring tasks for habits or repeated responsibilities.
- Add reminders for overdue or upcoming tasks.
- Add task categories such as School, Work, Health, and Personal.
- Add a streak tracker for completing tasks across multiple days.
- Add a daily review section where users confirm what they finished.
- Add filtering for All, Today, Overdue, Completed, and Focus tasks.
- Add an "off-task warning" message when too many tasks are overdue.

## High Scope Features

- Add a focus mode that hides non-priority tasks until the current task is finished.
- Add timed work sessions with breaks, similar to a Pomodoro timer.
- Add accountability reports showing weekly completion rates and missed tasks.
- Add goal tracking that connects small tasks to larger long-term goals.
- Add a lock-in mode where a task cannot be dismissed without giving a reason.
- Add calendar integration for scheduling tasks across days.
- Add optional accountability partner sharing so another person can see progress.
